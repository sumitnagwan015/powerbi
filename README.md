# 📊 Store Sales Dashboard – Power BI

## 📌 Overview

This Power BI dashboard provides a comprehensive view of store performance, helping stakeholders monitor sales trends, customer behavior, and product performance. It is designed to support data-driven decision-making through interactive visualizations.

---

## 🎯 Objectives

* Track overall sales performance
* Analyze product-wise and category-wise sales
* Monitor customer purchasing patterns
* Identify top-performing regions and stores
* Enable quick insights through interactive filters

---

## 📁 Dataset

The dataset used in this dashboard includes:

* Order details (Order ID, Date, Customer ID)
* Product information (Category, Sub-category, Product Name)
* Sales metrics (Revenue, Quantity, Profit)
* Location data (City, State, Region)

---

## 📊 Key Metrics (KPIs)

* Total Sales
* Total Profit
* Total Orders
* Quantity Sold
* Profit Margin

---

## 📈 Dashboard Features

### 1. Sales Overview

* Displays total sales, profit, and order count
* Monthly and yearly trends

### 2. Product Analysis

* Sales by category and sub-category
* Top-selling products

### 3. Customer Insights

* Repeat vs new customers
* Customer contribution to revenue

### 4. Regional Performance

* Sales by region/state
* Map visualization for geographic insights

### 5. Time-Based Analysis

* Trends by month, quarter, and year
* Seasonal patterns

---

## 🎛️ Filters & Interactivity

Users can interact with the dashboard using:

* Date slicers
* Region/City filters
* Product category filters

---

## 🛠️ Tools & Technologies

* Power BI Desktop
* Data transformation using Power Query
* DAX (Data Analysis Expressions) for calculations

## 🚀 How to Use

1. Open the `.pbix` file in Power BI Desktop

3. Hover over visuals for detailed insights
4. Navigate between report pages (if multiple pages exist)
